# macOS Legacy 설정

이 디렉토리는 현재 기본 macOS 설정 흐름에서 제외한 오래된 설정과 선택 설정을 보관합니다.

- `iterm2/`: 예전 iTerm2 프로필 export와 설치 스크립트입니다. 현재 터미널 기준은 Ghostty/tmux입니다.
- `.skhdrc`, `.yabairc`: 예전 yabai/skhd 창 관리 설정입니다. 현재 창 관리 기준은 AeroSpace입니다.
- `raycast/exports/`: 참고용으로 남겨둔 오래된 Raycast export입니다.
- `raycast/raycast_command/`: 기본 활성 command set에서 제외한 큰 외부 Raycast extension과 iTerm 전용 command입니다.

필요할 때만 이 디렉토리에서 꺼내 사용하고, 기본 bootstrap 대상에는 포함하지 않습니다.
