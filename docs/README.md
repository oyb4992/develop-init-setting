# 문서

이 디렉토리는 프로젝트 전반의 보조 문서를 두는 공간입니다. 루트 `README.md`는 빠른 시작과 현재 구조를 설명하고, 이 디렉토리에는 더 긴 설치 기록, 문제 해결, 도구별 상세 설명을 추가합니다.

## 권장 문서 범위

- 설치 절차와 수동 복구 방법
- macOS/Windows별 차이점
- 터미널, 쉘, tmux, 에디터, 윈도우 매니저 설정 설명
- 로컬 서비스 운영 방법
- 정리/백업/마이그레이션 체크리스트

## 작성 규칙

- 실제 파일 경로를 기준으로 작성합니다.
- 개인 토큰, 계정 정보, `.env` 값은 문서에 남기지 않습니다.
- 외부 도구에서 자동 생성한 README는 가능하면 해당 도구 디렉토리 안에 그대로 둡니다.
- 구조가 바뀌면 루트 `README.md`의 디렉토리 구조도 함께 갱신합니다.
